Command Line Interface

Syntex can be used as a command line utility as well as a Python library. When used on the command line Syntex reads its input from stdin and prints its output to stdout.

Use the syntex --help flag to view the utility's command line help:

Usage: syntex [FLAGS]

  Renders input text in Syntex format into HTML. Reads from stdin and
  prints to stdout.

  Example:

      $ syntex < input.txt > output.html

Flags:
  -d, --debug       Run in debug mode.
      --help        Print the application's help text and exit.
  -p, --pygmentize  Add syntax highlighting to code samples.
      --version     Print the application's version number and exit.

Syntex can use the Pygments package if it's available to add syntax highlighting to code samples - use the --pygmentize flag to turn this feature on.

Note that Pygments is installed automatically when Syntex is installed using pip.
